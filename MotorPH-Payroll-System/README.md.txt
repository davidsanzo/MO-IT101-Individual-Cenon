# MotorPH Payroll and Attendance System

## Overview
This project demonstrates a simple payroll system designed to manage employee records and compute weekly salary.

## Features
- Employee information storage
- Salary computation
- Weekly payroll summary

## System Components
1. Employee Details
2. Salary Calculation
3. Payroll Processing
4. Weekly Salary Summary
